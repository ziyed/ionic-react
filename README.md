# PhotoSnap
This snap taking application helps you to build your own Photo Snap Gallery. 
Where ever you are just open the app and capture a quick snap. 
That's it and enjoy the wonderful feature, SMILE.


# Technology
- Ionic Framework 5
- ReactJs
- Firebase Hosting
- Prograssibe Web App
- iOS 
- Android
